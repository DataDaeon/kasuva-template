Magento Installation

Thank you very much for purchasing! In this document you will find all the information about how to setup and configure this theme in your Magento Installation.

System Requirements

Unicase only supports Magento Community Edition 1.9.x, 1.8.x, 1.7.x which can be downloaded from http://www.magentocommerce.com/download.

It does not support Magento Enterprise Edition and Magento GO. Before downloading Magento, make sure you meet the system requirements and test the compatibility of your server.

For help with installing Magento, please refer to the following links:

  • Magento System Requirements
  • Server Compatibility Check
  • Magento Installation Guide

Quickstart Installation

In this theme package we included sample data with settings of demo version. Follow the instructions below in order to install exactly the same store as on demo version:

  • Uploading Quickstart Files

    Log into your hosting space via a FTP software. ie, FileZilla.

    Unzip Theme Package > Quickstart Package > Unicase_quickstart_v1.0.zip and upload all directories and files to your root directory.

  • Import Database

    Create a new database, and import loacated Theme Package > Quickstart Package > Database > Unicase_demo_v1.0.zip zip file into your database.

  • Open browser and proceed to the folder where magento was unzipped and load default Magento setup wizard
  • Follow the wizardâ€™s instructions and finish magento installation
  • After installation clear magento cache go to System > Cache Management and Reindex magento data go to System > Index Management

Theme Installation

Uploading Theme Files

Before you begin the installation process, we recommend to backup your previous Magento installation and database.

  • Unzip downloaded folder Theme Package > Theme Files > Unicase Theme.zip
  • Upload / Copy all files / folders from folder “Magento Files” to your Magento root directory, if you are using Mac, please make sure you are merging and not replacing Magento directory.
  • Disable compilation mode in System > Tools > Compilation
  • Warning! : After upload all the files and folders please don't forgot to remove all cache and session file from ' var > cache > ' and ' var > session > ' folder and relogin to the admin after logout.
  • Log out from magento admin panel and log in again.

Theme Setup

  • Login to the Magento Admin Panel.
  • Go To: System > Configuration Click on Design on the left menu
    • Under Package Tab put perfectusinto current package name
    • Next Go to the Theme Tab and put Unicaseinto Templates, Skin, Layout, Default input boxes.
    • Under Header tab put “images/logo.png” into 'Logo image Src' Field
    • You can also enter the Image alter text and Default Welcome Text
  • Scroll back to the top of the page and click the Save Config button
  • Next, go to System > Cache Management from the top navigation menu. You will need to clear the store cache and all cookies.
  • Click the following buttons (in orange): Flush Magento Cache, Flush Cache Storage, Flush JavaScript/CSS Cache, and Flush Catalog Images Cache (optional). Now you are ready to add content to your website.

Mega Menu Configuration

Please follow below step to set Megamenu. For more Megamenu info please go to Megamenu Extension.

  • Go to:Catalog > Manage Categories and click on your categories
  • Now you can see there tab with the name "Megamenu" , Click on that tab to see options.
  • Set the following option under 'Megamenu' Tab

    • Show in Menu > Here you can set the category which display on frontend Navigation Megamenu.
    • Show in Verticle Menu > Here you can set the category which display on frontend Verticle Megamenu
    • Menu Type > Here you can set menu style like 'Default', 'Full Width', 'Dropdown Style' and 'Custom' for each category.
      • Default > when you set default it will affected from Unicase Extension > Megamenu > Default Menu Type
      • Full Width > Display Full Width Menu of the site.
      • Dropdown Style > Display Dropdown style menu.
      • Custom > to set custom width menu.
    • Dropdown Custom Width > Here you can set the Menu width like "800px", Applicable only Menu Type = Custom.
    • Show Columns > Here you can set number of columns display in Megamenu. Applicable only Top Level Sub Categories.
    • Category Labels > Here you can set label of category which display on frontend menu.
    • Icon (Font Awesome) > Here you can set Category Icon in Verticle Megamenu . Ex:"fa fa-blind", This fields applicable only for Top Level Categories.
    • Left Block Width > Here you can set width of left block in based on fluid width like 1 to 12.
      Make sure the sum of 'Left Block Width' , 'Right Block Width' and 'Category Block Width' is '12'.
      Exa: In our 'Woman' category we set Right Blow Width is '4' and Category Block Width is '8'. This field applicable only for Top Level Categories.
    • Left Block > Here you can add HTML and CMS Block which you have created by going to CMS > Static Blocks to display on left side of menu popup. This field applicable only for Top Level Categories.
    • Right Block Width > Here you can set width of right block in based on fluid width like 1 to 12.
      Make sure the sum of 'Right Block Width' , 'Left Block Width' and 'Category Block Width' is '12'.
      Exa: In our 'Woman' category we set Right Blow Width is '2', Left Blow Width is '2' and Category Block Width is '8'. This field applicable only for Top Level Categories.
    • Right Block > Here you can add HTML and CMS Block which you have created by going to CMS > Static Blocks to display on Rigth side of menu popup.
    • Top Block > Here you can add HTML and CMS Block which you have created by going to CMS > Static Blocks to display on Rigth side of menu popup. This field applicable only for Top Level Categories.
    • Bottom Block > Here you can add HTML and CMS Block which you have created by going to CMS > Static Blocks to display on Rigth side of menu popup., This field applicable only for Top Level Categories.

Blog Configuration for Go to Perfectus > Blog

After enabled blog you want to display blog slider in CMS Page use below code and set block_count to display maximum number of post display in slider

{{widget type="blog/slider" blocks_count="6"}}

Below list of configuration for display blog pages.


Enable > Enable or Disable blog.

Route to Blog > You can set blog url to own name, Must not contain any slashes. "news" makes the blog accessible from domain.com/news. Leave blank for default value.

Blog Title > You can set blog title.

Meta Keywords > You can set blog page meta keywords.

Meta Description > You can set blog page meta description.

Page Layout > Here you can set page layout to "1 column", "2 columns with left bar", "2 columns with right bar", "3 columns".

Date Format > Here you can set date format "Full", "Long", "Medium" and "Short".

Show Breadcrumbs > Hide or Show breadcrumbs.

Summary Character Count > Number of characters in a post to be displayed before it is truncated and the Read More link is added. 0 for unlimited.

Use Short Content Field > If enabled, additional "Short Content Field" is taken instead of truncated main post.

Parse CMS Tags in Post Contents > You can parse the tag in post contents.

Posts per Page > You can set number of post to display post per page.

Show Bookmarks in Lists > Set "yes" to show bookmarks in Lists.

Show Bookmarks in Posts > Set "yes" to show bookmarks in Posts.

Show Tags in Posts > Set "yes" to show tags in Posts.

Show Tags in Lists > Set "yes" to show tags in lists.

Use Categories ID in Posts URLs > Set "yes" to show categories id in posts.

Default Posts Sorting > You can set sort order for the post to "newest first" or "older first".


Menus & Links > Here you can set enable or disabled menus and links.


Comments > In thes options to set commetns options.

ReCAPTCHA > In thes options to set ReCAPTCHA.

RSS Feed > In thes options to set RSS Feed.
